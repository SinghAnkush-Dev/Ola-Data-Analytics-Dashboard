# Ola Ride Booking Data Analytics

This project analyzes Ola ride booking data to uncover valuable business insights such as ride demand, revenue trends, cancellation rates, customer ratings, and peak booking hours.

## Tools Used
- SQL
- Python (Pandas, Matplotlib)
- Excel
- Power BI

## Key Insights
- Total rides and revenue analysis
- Peak booking hours identification
- Vehicle type performance
- Payment method analysis
- Ride cancellation trends
- Customer and driver rating analysis
